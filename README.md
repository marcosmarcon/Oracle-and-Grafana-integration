## Getting Started

First it is necessary to install Python and Python library cx_Oracle.
### Installation

1. Clone the repo
```sh
git clone https://github.com/marcosmarcon/Oracle-and-Grafana-integration
```
2. Install packages
```sh
python -m pip install cx_Oracle --upgrade pip
```
